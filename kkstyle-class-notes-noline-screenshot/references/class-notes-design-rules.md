# Class Notes Design Rules

## Core Mood

The visual language should feel:

- smart
- handwritten
- study-like
- organized
- friendly

It should not feel:

- chaotic scrapbook
- corporate slide
- futuristic UI
- luxury editorial

## Palette

Use this order of importance:

1. `#fffef7` / `#fbf8ee` as notebook-paper background
2. `#48434d` as softened pen ink
3. `#4b73ff`, `#ff7aa8`, `#ffe76a`, `#7de3b1` as highlighter accents
4. light blue or gray ruled lines as secondary structure

## Layout Bias

The default `3:4` poster should feel like a clean note page:

- one handwritten main title
- one highlighted takeaway strip
- two main note columns
- one lifted footer note area

The title should feel like bold marker writing:

- keep it large and slightly irregular in spirit
- let highlighted summaries sit close below
- maintain clean spacing between note blocks

## Date Label

Every poster should include a date label:

- place it near the upper-right
- use `YYYY年MM月DD日`
- keep it compact and clearly readable

## Contact Cluster

Every poster should include the bundled contact cluster:

- use `assets/contact-code.png` as the QR card
- use `assets/kk-character.jpeg` as the IP card
- place the QR to the left of the IP
- keep both cards the same size
- keep `合作交流` and `是卡卡呀` as compact captions below

## No-Line Rule

- avoid visible black outlines
- separate sections with fill, shadow, blur, or spacing first
- if a separator is needed, keep it low-contrast and secondary

## Final Check

- the classroom notes feeling is obvious
- the date label is visible and correctly formatted
- the QR and Kaka IP are present
- the two contact cards are the same size
- the page still reads clearly as an information poster
