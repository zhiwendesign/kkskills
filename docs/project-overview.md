This repository packages a cute, bold, mascot-like card style into an agent-friendly skill.

## Visual Direction

The style is intentionally playful instead of editorial:

- bright cobalt blue backgrounds
- soft white card surfaces
- black high-contrast outlines
- pixel-grid framing and sticker accents
- a seal-like top silhouette that makes the card feel character-driven
- a fixed upper-right date chip in `YYYY年MM月DD日`
- a fixed lower-right contact cluster with QR + Kaka IP

The repository now includes a second direction for warmer layouts:

- scrapbook paper backgrounds
- taped note cards and layered paper sheets
- cozy editorial collage rhythm

It also includes a third direction for sharper layouts:

- magazine-like mastheads and cover lines
- restrained off-white / black / deep red contrast
- cleaner editorial structure with stronger hierarchy

And a fourth direction for game-like layouts:

- dark handheld-style HUD backgrounds
- chunky pixel borders and status cards
- more playful retro console energy

It now also includes three more specialized directions:

- classroom notes and highlighted study sheets
- minimal tech launch posters
- comic-book style cover posters

And it now also includes a diagram-driven explainer direction:

- whiteboard-like infographic boards
- modular steps, node cards, and connector hints
- a more visual teaching / walkthrough rhythm

For each of the eight base directions, the repository now also includes a matching no-line variant:

- same visual identity, but without visible black outlines
- softer boundaries built from fill, shadow, spacing, and low-contrast separators
- better suited to cleaner, lighter, more modern poster outputs
