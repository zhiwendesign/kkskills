# Poster Comic Design Rules

## Core Mood

The visual language should feel:

- loud
- bold
- playful
- punchy
- readable

It should not feel:

- corporate
- minimal
- scrapbook handmade
- soft luxury

## Palette

Use this order of importance:

1. `#ffef55`, `#ff5f5f`, `#3a88ff`, `#fffaf0` as main colors
2. `#37363d` as softened soft text and ghost separators
3. use halftone dots and blocks sparingly

## Layout Bias

The default `3:4` poster should feel like a comic cover:

- one huge title
- one bright highlight badge near the upper-right
- one speech-bubble summary
- two main info panels
- one lifted footer note area

## Date Sticker

- keep it near the upper-right
- use `YYYY年MM月DD日`
- make it feel like a sticker, not a plain label

## Contact Cluster

- use `assets/contact-code.png` and `assets/kk-character.jpeg`
- keep both cards the same size
- place the QR to the left of the IP
- keep `合作交流` and `是卡卡呀` as compact captions below

## No-Line Rule

- avoid visible black outlines
- separate sections with fill, shadow, blur, or spacing first
- if a separator is needed, keep it low-contrast and secondary

## Final Check

- the comic poster feeling is obvious
- the date sticker is visible and correctly formatted
- the QR and Kaka IP are both present
- the two contact cards are the same size
- text remains readable despite the louder design
