# Layout Presets

Use these fixed canvases for kkstyle cards.

| Ratio | Typical use | Canvas size |
| --- | --- | --- |
| `3:4` | Main portrait card | `1500 x 2000` |
| `4:3` | Landscape summary card | `2000 x 1500` |
| `1:1` | Social square post | `1800 x 1800` |
| `16:9` | Video cover or wide banner | `1920 x 1080` |
| `9:16` | Story or vertical cover | `1080 x 1920` |

## Notes

- Start from the exact target canvas.
- Keep extra decoration near the frame or corners, not over the body text.
- Portrait layouts should keep the mascot silhouette stronger.
- Wide layouts should reduce paragraph length and increase sticker-style side labels.
