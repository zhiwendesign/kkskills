# Minimal Tech Design Rules

## Core Mood

The visual language should feel:

- precise
- modern
- clean
- restrained
- premium

It should not feel:

- overly playful
- scrapbook handmade
- comic loud
- dark cyberpunk by default

## Palette

Use this order of importance:

1. `#f5f7fb` / `#ffffff` as main background
2. `#313b49` / `#465468` as text and softly separated panels
3. `#5ac8fa` / `#66f2ff` as small accent
4. `#d9e1ef` / `#b7c5d8` as separators and muted fills

## Layout Bias

The default `3:4` poster should feel like a launch page:

- one dominant title block
- one compact metric card near the upper-right
- one lead paragraph
- two main content columns
- one lifted footer note area

## Date Label

- keep it near the upper-right
- use `YYYY年MM月DD日`
- make it crisp and compact

## Contact Cluster

- use `assets/contact-code.png` and `assets/kk-character.jpeg`
- keep both cards the same size
- place the QR to the left of the IP
- keep `合作交流` and `是卡卡呀` as compact captions below

## No-Line Rule

- avoid visible black outlines
- separate sections with fill, shadow, blur, or spacing first
- if a separator is needed, keep it low-contrast and secondary

## Final Check

- the page reads as minimal-tech immediately
- the title is primary
- the metric card is clearly visible
- the QR and Kaka IP are both present
- the two contact cards are the same size
